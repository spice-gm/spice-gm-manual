一、简介

简介

本手册的主要内容是通过改造Spice项目(Protocol, Server, GTK Client, Web Client)，使之支持基于SM算法的TLS加密传输 (参考 RFC 8998 & GB/T 38636-2020)

注意： 本项目在WSL2环境下完成，手册中包含有关WSL虚拟化环境配置的章节。
